PROFILE IMAGE (for hero)
------------------------

File name (recommended): public/images/profile.jpg
Alternate accepted: public/images/profile.png

Recommended size: 1200 x 1200 px (square). Optimized for web:
 - width: 800-1200px is fine
 - file format: JPG (high quality) or PNG (if transparency)
 - compress to ~100-300 KB for faster load
 - crop to a square, center the face or main subject

To add your photo:
 - Save as `profile.jpg`
 - Place in the project root under: public/images/profile.jpg
 - Restart dev server NOT required; Next.js watches /public

If you don't add profile.jpg, the site will show profile-placeholder.svg automatically.
